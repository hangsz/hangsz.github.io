能避免中年危机或许是个伪命题

发表： 2017-03-09 | 更新： 2017-04-12 | 专题： 杂谈 | | 阅读次数：

版权声明：文章为博主原创，转载请注明出处。

作为即将入职华为的人，看着华为最近搞出的大新闻，也是百感交集。最近思考了这个中年危机这个问题，我认为避免中年危机的关键就是：考虑历史的进程 自我的不断奋斗。

1.考虑历史的进程

危机由外部环境施加，想要避免危机必须知道危机从何而来。

进程一 第二个生育高峰出生的人都逐步参加工作，劳动市场上新增大量优质劳动要素供给。

看两张图，

1960 ~ 2010 年历年初始的人口数量[1]

2001年到2015年全国高校毕业生人数[2]

由上图可以看到，第二个生育高峰峰值大约在1987-1992范围内，对应劳动市场上即将工作和工作两三年的新员工。按理说，中国人口红利快要没得吃了，供给增量减少，对劳动要素的需求应该增加才对。其实不然，这句话正确的前提是市场需求增量不减，然而现实并非如此。

进程二 全国经济增长率不断下降，导致对新增劳动要素的需求减少。

中国GDP增长率

明显看出，GDP增长率在不断下降，而GDP增长率和就业正相关。具体原因来看看李总理怎么说[3]：

我为什么上来要跟大家讲GDP呢？同志们可能会问，说我们不能片面强调GDP。这个话说得不错，但我们毕竟是发展中国家，发展还是我国解决一切问题的基础和关键。更重要的是，我们关注GDP，其实关注的是就业。过去，我国GDP每增长1个百分点，就会拉动大约100万人就业。经过这几年经济结构的调整，尤其是随着服务业的加快发展，目前大概GDP增长1个百分点，能够拉动130万、甚至150万人就业。我请人力资源和社会保障部和有关方面反复测算，都认为要保证新增就业1000万人、城镇登记失业率在4%左右，需要7.2%的经济增长。我们之所以要稳增长，说到底就是为了保就业。

7.2%的增长率，能保1000万就业，而近期将要或者已经参加工作的新员工的同龄人约2000多万。即使单从高校毕业生800万角度来看，6.7%又能保证多少比较高端的职位呢？
面对经济下行的压力，怎么办？要稳增长、保就业，我们有两种选择。一种选择是扩大财政赤字，增加货币供应量。
根据经济学第十大原理：增发货币短期内可以增加就业，长期导致通胀。此策略虽有用，但是结果大家都知道了，货币都流某蓄水池了，年轻人深受其害。所谓中年危机，最大的危机就在这里，吹高的泡沫风险都转嫁到老百姓头上，或许是因为中国老百姓的信用太好。
经济的高速增长让大多数人的状况变得更好而掩盖了很多矛盾，华为34岁“裁员”新闻不过是逐渐揭开的残酷现实的一角而已。

好了，我想大家对上述内容也有模糊的认知，不过看到具体数据更能加深对社会残酷形势的认识。不过，认识到危机的历史产生必然性只是我想说的一个方面的，另一方面我想强调的是要认识到能避免中年危机或许只是个伪命题，认识到人力有时而穷，希望诸位等到自己面临危机的那一天能够做到原谅自己的无能无力。

2.自我的不断奋斗

普通人虽无法左右国家、行业大势，但随遇而安也非年轻人所需之道。
摘两句名言，诸位共勉：

激水之疾 ，至于漂石者，势也。
《孙子兵法·势篇》

根据这句话，我给年轻人的第一个建议是：学会借势，这叫危机对冲。

  • 风口飞的猪或许强于地上的老虎。关注市场热门行业，依托本行业积累积极寻求转型，上升行业新手红利不一定比稳定行业老手收益差。以我个人为例，今年再转入计算机行业虽然有点晚，不过红利我还是吃到了。
  • 人生最重要的是资产配置。几年前借到房地产大势的人，不知道什么叫中年危机。目前面对高企的房价，年轻人必须好好体会国家的政策对此产业的影响，观察好形势再上车。

凡战者，以正合，以奇胜，故善出奇者，无穷如天地，不竭如江河。
《孙子兵法·势篇》

根据这句话，我给年轻人的第二个建议是：有备无患，这叫危机保险（去开个公司对现在被华为吓的“风声鹤唳”的年轻人卖危机保险保证赚钱）。
无论什么时候，努力奋斗的同时都要留一手或者说有副业，总之要有Plan B。有正有奇、正奇循环才能职场长青。此外，建议大家读一下大咖 @田渊栋 的这个回答：面对飞速发展的人工智能，我们应该如何调整对孩子的教育？ - 儿童教育 - 知乎 ，思维的变化一定要跟上世界的变化。

首先，世界变化会更快，竞争会更激烈。铁饭碗不会有了，一个工作做一辈子也很少见了，“努力一把一辈子享福”，“一招鲜吃遍天”等等观念，都会被扫进历史的垃圾堆。如果仍然抱有这样的观念，可能需要好好地改变一下。这个世界，已经不是以前的样子了。
