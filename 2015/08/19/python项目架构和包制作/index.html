3. 包制作命令 setup.py 命令

进入项目根目录，运行制包命令，python setup.py --help-command 查看详细。

  • python setup.py bdist_egg # 生成类似 bee-0.0.1-py2.7.egg，支持 easy_install
  • python setup.py sdist # 生成类似 bee-0.0.1.tar.gz，支持 pip
  • python setup.py build #编译
  • python setup.py bdist_wininst # Windows exe python setup.py bdist_rpm # rpm
